# Frontend Development Guidelines

> Best practices for frontend development in this project.

---

## Overview

This directory contains frontend conventions for the Platform Template Vue admin app. These rules describe the current codebase and should be read before changing frontend code.

## How to Use These Guidelines

Before changing frontend code, read this index and the files matching the touched area. For route/menu/API changes, also read backend database and logging/error specs as needed.
