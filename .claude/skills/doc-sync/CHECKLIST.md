## 🔍 检查项

### 1. 数据库相关（如修改了 schema.prisma）

- [ ] **seed.ts** - 是否需要初始化数据？

  - [ ] 修改了现有模型的必填字段
  - [ ] 添加了枚举值需要示例

- [ ] **version/v*.md** - 数据库设计文档是否更新？
  - [ ] 表结构说明

- [ ] **AGENTS.md** - 数据库架构章节是否需要更新？

- [ ] **version/v*.md** - API 文档：
  - [ ] 接口路径是否正确？

- [ ] **AGENTS.md** - 项目结构：
  - [ ] 新增模块是否在目录树中？

```

backend/src/app.module.ts       # 根模块
version/v*.md                    # 版本计划
AGENTS.md                        # 项目文档
frontend/src/router/index.ts    # 前端路由
.claude/rules/*.md              # 规范文档
```
